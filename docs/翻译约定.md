- 英文原文使用 variable 指代了变量、函数、对象等多种类型，并不仅限于”变量“，在翻译中我们遵循原文这一约定
- `{}`翻译为花括号
- 每节课后的小测试暂时不翻译，全部完结后统一翻译并解析
- 尽管自然语言中的句法和语法是不同的，在翻译计算机文档时，[[句法(syntax)]] 通常也被译为语法，本项目也采取该做法。
- `operator` 一般翻译为操作符或运算符，本文统一翻译为运算符。
- `specifiers`翻译为修饰符
